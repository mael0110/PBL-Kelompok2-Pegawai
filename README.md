# Vue JS hit Api

Fokus ke folder `src/` aja karena semua codingan nanti ada didalam folder itu, sama perhatikan file `kategori.vue` aja untuk front end nya, dan `service/kategori.js` untuk kode hit api nya.

## Cara menggunakan

Buka terminal

> clone project

```terminalOutput
git clone https://github.com/akufarish/vue-hit-api.git
```

> akses folder hasil clone

```terminalOutput
cd vue-hit-api
```

> install dependency yang diperlukan

```terminalOutput
bun i
```

> jalankan project

```terminalOutput
bun dev
```

> jalankan json-server pada terminal terpisah

```terminalOutput
bunx json-server db.json
```

Sekarang project bisa diakses pada localhost:5173
